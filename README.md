# 91海角社区-吃瓜-爆料海角

第四十九章 成为棋局中的一枚小卒，为其冲锋陷阵
　　终于，在秦曼云翘首以盼的注视下，孟君良缓步而来。
　　整个酒楼顿时安静，只有孟君良的声音缓缓传出。
　　“玉帝传旨，即着云部众神，分头请三清、四御、五老、六司、七元、八极、九曜、十都、千真万圣，来此赴会，同谢佛恩。又命四大天师、九天仙女，大开玉京金阙、太玄宝宫、洞陽玉馆，请如来高坐七宝灵台。调设各班座位，安排龙肝凤髓，玉液蟠桃……”
　　一个奢华到极点的宴会在众人面前缓缓铺展开，虽然只是在故事中，但他们大气都不敢喘。
　　出席宴会的一个个神仙的名字以及职务就听得他们头晕目眩，几乎失去了思考的能力。
　　至于餐桌上的龙肝凤髓那更是想都不敢想。
　　那可是龙和凤啊，上古神兽，刚一出生就是仙人境界，在修仙者眼中不可侵犯，遇到了只有跪拜的份，哪里还敢想着吃。
　　然而，在这天宫之上，龙肝凤髓居然成了一盘菜。
　　可怕，太可怕了！
　　秦曼云在震惊的同时，也不忘展开联想，思考其中蕴含的深意。
　　书生既然说这里需要考验自己的悟性，那自己一定要注意。
　　她皱着眉头，屏气凝神。
　　天宫是在庆祝孙悟空被镇压，这和我们修仙界击败了强大的敌人后庆祝何其相似。
　　失败者被压在最底层，不得翻身，胜利者高高在上，举杯庆祝。
　　至于龙和凤，这些在仙人的眼中跟普通的妖精有什么区别呢？
　　原来仙人除了实力强大外，其实跟修仙者并没有什么不同。
　　这是在暗示什么呢？
　　秦曼云继续往下听。
　　书生语气平稳，情节一点点推进。
　　很多人只当是一个故事，但是有人已经渐渐皱起了眉头，带着一丝深思。
　　总感觉这背后隐藏着惊天大秘密。
　　孙悟空被镇压，故事开始围绕着佛门展开。
　　“诸菩萨闻言，合掌皈依，向佛前问曰：“如来有哪三藏真经？”如来回：“我有法一藏，谈天；论一藏，说地；经一藏，度鬼；三藏共计三十五部，该一万五千一百四十四卷，乃是修真之径，正善之门。
　　我待要送上东土……寻一个善信．教他苦历千山，远经万水，到我处求取真经，永传东土，劝他众生，却乃是个山大的福缘，海深的善庆、谁肯去走一遭来？””
　　这，这，这……
